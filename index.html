Scala语言基础

Scala语言基础概念及用法

##函数的参数：函数参数值得求值策略（call by value、call by name）

###函数参数值得求值策略
call by value：对函数的实参求值，仅求一次
举例：使用：
def test1(x:Int,y:Int):Int = x + x
调用：test1(3+4,8)
call by name：函数的实参每次在函数体内部被调用到，都会求值
举例：使用： =>
def test2(x:=> Int,y:=> Int):Int = x + x
调用：test2(3+4,8)
注意：执行的过程不一样的
稍微复杂一点的例子：
x是call by value y是call by name
def bar(x:Int,y: => Int): Int =1
定义个死循环

###数组
import scala.collection.mutable.ArrayBuffer
//Scala数组

//类型一：定长数组 Array
val a = new Array[Int](10)
val b = new Array[String](3)
var c = Array("Tom","Mary","Mike")

//类型二：变长数组 ArrayBuffer
val d = new ArrayBuffer[Int]()
//往变长数组中添加元素
d += 1
d += 2
d += 3
d += 4
d += 5

//一次添加多个元素

d.trimEnd(2) //去掉尾部的两个元素


//把ArrayBuffer --> Array
d.toArray

//遍历数组
//foreach 没有返回值；map有一个返回值

d.map(println)
